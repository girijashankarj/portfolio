## Overview

**portfolio** is a static personal portfolio site that showcases professional background, projects, writing, certifications, and consulting services. It's built with HTML5, Tailwind CSS, and Bulma — no build step required. Deployed on GitHub Pages at [girijashankarj.github.io/portfolio](https://girijashankarj.github.io/portfolio/).

Focus areas include architecture, AWS, AI enablement, and React/Node.js development.

---

## Features

- **Dark/Light theme** — toggle with localStorage persistence
- **Scroll progress indicator** — visual reading progress bar
- **Smooth scroll navigation** — anchor-based section navigation
- **Intersection Observer animations** — fade-in effects on scroll
- **Responsive design** — mobile-first layout
- **PWA-ready** — web manifest and favicon set included
- **No build step** — static HTML served directly

## Quick Start

```bash
git clone https://github.com/girijashankarj/portfolio.git
cd portfolio
```

Open `index.html` directly in your browser — no build step required.
